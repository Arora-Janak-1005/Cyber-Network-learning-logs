## Metasploit Framework:
- a framework used to find exploit and validate security vulnerabilities 
- basically it is a library  of prewritten exploits for different vulnerabilities , in which you can also modify the payload , it is a very strong tool used in security testing.
## basic using steps:
- sudo msfconsole
- seacrh vulnerbility
- use exploit
- show options(includes the info about the target the  interface,etc)
- set target options
- exploit command to run
